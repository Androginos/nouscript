# NouScript YouTube Cookies Eklentisi

YouTube videoları için cookie iznini otomatik almak için Chrome eklentisi.

## Kurulum

1. Bu `extension` klasörünü indirin (veya repoyu clone edin)
2. Chrome'da `chrome://extensions` adresine gidin
3. Sağ üstten **"Geliştirici modu"**nu açın
4. **"Paketlenmemiş öğe yükle"** butonuna tıklayın
5. `extension` klasörünü seçin

## Kullanım

1. NouScript sitesine gidin
2. **"YouTube için cookie izni ver"** butonuna tıklayın
3. Eklenti YouTube cookie'lerinizi otomatik alır
4. YouTube linkini yapıştırıp işleme başlayın

Cookie'ler sadece butona tıkladığınızda alınır ve yalnızca o istek için kullanılır.
